# launchpad-portfolio
Launch Pad portfolio showcasing IT, fundamentals, organization, and documentation skills. 
Launchpad Portfolio – Lab 01
Overview
This repository is part of Lab 01: Setting Up Your Engineering Log.
It serves as a foundational technical portfolio demonstrating the use of GitHub, command-line tools, and structured documentation practices.

Objective
The objective of this lab is to:

Practice working with GitHub repositories
Use the Command Line Interface (CLI) for version control
Document technical work in a clear and professional format
Establish a repeatable workflow for future labs
Repository Structure
Lab-01.md – Engineering log documenting the lab activity
configs/ – Configuration-related files
scripts/ – Scripts created or used during labs
diagnostics/ – Diagnostic outputs and notes
proof/ – Screenshots and evidence of CLI usage
README.md – Project overview and documentation
Workflow (CLI-Based)
All work in this repository was completed using the Command Line Interface (CLI):

Repository cloned using git clone
Files edited locally
Changes staged using git add
Commits created with descriptive messages
Changes pushed to GitHub using git push
This workflow ensures proper version control and traceability.

Proof of Work
Terminal screenshots are included in the proof/ directory
Git commit history shows multiple commits created via CLI
No web-based uploads were used for core file modifications
